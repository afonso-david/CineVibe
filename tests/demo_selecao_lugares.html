🎬 Demo - Sistema de Seleção de Lugares

📋 Funcionalidades Implementadas

  • Planta da sala gerada dinamicamente baseada na capacidade da base de dados
  • Layout inteligente: Pequena (10x15), Média (15x20), Grande/IMAX (20x18)
  • Visualização de lugares ocupados em tempo real
  • Pré-seleção automática de lugares em grupo consecutivo
  • Seleção interativa sem mensagens de alerta
  • Interface limpa sem letras de fileiras
  • Design premium com cores escuras e elegantes
  • Integração completa com a base de dados existente
  • Validação de lugares já reservados por sessão e data
  • Interface responsiva para desktop e mobile

🏗️ Estrutura do Sistema

1. Página de Reserva: Seleção da quantidade de bilhetes (1-6 ou personalizada)

2. Página de Seleção de Lugares: Planta da sala com pré-seleção automática de lugares em grupo

3. Interação: Usuário pode alterar a seleção clicando nos lugares desejados

4. Confirmação: Processamento da reserva na base de dados

💾 Base de Dados

Salas: 240 salas com capacidades definidas (100-400 lugares)

Tipos de Layout:

  • Sala 4DX: 100 lugares → Layout 10x15 (Pequena)
  • Sala IMAX: 350 lugares → Layout 20x18 (Grande)
  • Sala Normal: 400 lugares → Layout 20x18 (Grande)

Reservas: Sistema de controle por sessão_id + data_sessao
