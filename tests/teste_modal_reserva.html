🎬 Teste - Modal de Escolha de Reserva

Funcionalidade de Reserva

Este teste simula o comportamento quando um usuário não logado tenta fazer uma reserva.

Cenário: Usuário seleciona um horário e clica em "Reservar Agora" sem estar logado.

Fluxo Esperado

  1. Usuário clica em "Reservar Agora"
  2. Modal aparece com duas opções
  3. Opção 1: "Iniciar Sessão" → Redireciona para login e depois para reserva
  4. Opção 2: "Continuar como Convidado" → Vai direto para reserva
🎫

Como deseja continuar?

Para fazer a sua reserva, pode iniciar sessão ou continuar como convidado.
